## 🎯 DEPLOY NA VERCEL (GRATUITO)

### PASSO 1: Criar Conta na Vercel
1. Acesse: https://vercel.com
2. Clique em "Sign Up" (canto superior direito)
3. Cadastre-se com sua conta Google, GitHub ou email
4. **IMPORTANTE**: Use o mesmo email que você usa no GitHub

### PASSO 2: Conectar com GitHub
1. Após criar conta, clique em "Import Project"
2. Escolha "From Git Repository"
3. Conecte sua conta do GitHub (se ainda não conectou)
4. Procure pelo repositório: `emporio-demo`
5. Clique em "Import"

### PASSO 3: Configurar Deploy
A Vercel vai detectar automaticamente que é um projeto Next.js. As configurações já estão certas:

- **Framework Preset**: Next.js (automático)
- **Root Directory**: `./` (automático)
- **Build Command**: `npm run build` (já configurado)
- **Output Directory**: `dist` (já configurado)

### PASSO 4: Deploy
1. Clique em "Deploy"
2. Aguarde o build (cerca de 2-3 minutos)
3. **🎉 PRONTO!** Seu site estará online!
